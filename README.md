# @byjohann/utils

A collection of utility functions that I use across my JavaScript and TypeScript projects.

## Installation

Run the following command to add `tokenx` to your project.

```bash
# npm
npm install -D tokenx

# pnpm
pnpm add -D tokenx

# yarn
yarn add -D tokenx
```
